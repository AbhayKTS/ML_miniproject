# Chhaya Platform

## Overview
Chhaya is a multi-modal creative intelligence platform for text, image, and audio. It adapts over time through a persistent creative memory and feedback loops.

## Structure
- `frontend/`: React Vite Application (UI & Client Logic)
- `backend/`: Node.js Express API (Business Logic & AI Engines)
- `docs/`: System Architecture and Project Documentation

## Quick Start
To get started with the project locally:

**Backend:**
```bash
cd backend
npm install
npm run dev
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```
